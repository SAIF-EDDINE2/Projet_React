## Application de live chat développée en React, Socket.io et MongoDb

# Le fonctionnement de l'pplication

# Côté Serveur
- cd server
- npm install
après générer une clé JWT et rajoue la dans le fichier .env
- JWT_SECREAT_KEY = ""
et l'url de votre base de données MongoDb 
- MONGODB_URI= ""

# Côté Client
- cd client
- npm install
- npm start
